# PawLux — Pet Spa Landing Page

Landing page de una sola página (one-page) para **PawLux**, marca ficticia de spa y estética premium para mascotas en Los Ángeles, CA. Proyecto de práctica enfocado en diseño de conversión: presentación de servicios, galería, testimonios y formulario de reservación de citas.

**Demo en vivo:** https://damian7777777777.github.io/petShop/

## Características

- Diseño responsivo (mobile-first) con animaciones al hacer scroll.
- Navegación por anclas: Services, About, Gallery, Reviews, Book Now.
- Contadores animados (mascotas atendidas, años de experiencia, % de reseñas 5 estrellas).
- Carrusel/marquee de servicios en movimiento continuo.
- Tarjetas de menú de servicios con precios y etiquetas destacadas (Most Popular, Premium, Best Value).
- Sección "Our Story" con sello de certificación y badges de confianza.
- Galería de transformaciones ("Glow-Up Gallery").
- Carrusel de testimonios con calificación por estrellas.
- Formulario de reservación de citas con validación y confirmación visual al enviar.
- Footer con información de contacto, horarios y enlaces a redes.

## Stack técnico

- HTML5
- CSS3 (animaciones, transiciones, diseño responsivo)
- JavaScript (interactividad de formulario, contadores animados, scroll effects)
- Hospedado en GitHub Pages

## Cómo correrlo localmente

```bash
git clone https://github.com/Damian7777777777/petShop.git
cd petShop
# Abre index.html directo en el navegador,
# o sirve la carpeta con un servidor local:
npx serve .
```
